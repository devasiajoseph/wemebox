Wemebox is a hybrid content management system that sits right in between Wordpress and Hugo.

After working with some popular content management systems and static site builders we realized how painful it is to create and maintain a website without going through tons of guides and setup. We wanted something simple and an application which worked out of the box.

Wemebox is a product for people who want to create and deploy websites faster.

Goals

  1. Simple

    It will take only a minute to get started with Wemebox

  2. Portable

    Wemebox is extremely portable if you need to move your website into a new server. Just copy the directory to the system and it will work seamlessly like before.

  3. Lightweight

    Wemebox is optimized to use very less memory. Wemebox can run on a system with few MBs of RAM
